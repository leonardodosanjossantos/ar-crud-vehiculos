## Requirements
* [JAVA 11+](https://www.java.com)
* [Maven 3.8+](https://maven.apache.org)
* [Docker 20.10+](https://www.docker.com)
* [Docker-compose 1.27+](https://www.docker.com)
  * [MongoDB 4.4+](https://www.mongodb.com)

## Getting started

```bash
https://github.com/leonardodosanjossantos/ar-crud-vehiculos.git
cd ar-crud-vehiculos
```

## Build

```bash
mvn clean install
```
## Environment

```bash
docker-compose up -d --build
```
